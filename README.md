# Dota 2 "MMR, winrate, heroes tracker"
Простое веб-приложение на Flask, которое отслеживает статистику игроков в Dota 2:
MMR, винрейт, любимые герои, график активности и история поисков.

# Возможности

  Поиск по Steam ID
 
  Отображение MMR (соло/компетитивный)
  
  Winrate за последние 20 матчей
  
  Топ-3 героев с иконками
  
  График winrate за последние 30 дней
  
  История недавних поисков (сохраняется локально)
  
  Красивый адаптивный интерфейс с Dota 2-тематикой
  
  Поддержка фонового изображения

 # Технологии 
 
Backend: Python + Flask

API: OpenDota (бесплатный, без ключа)

Frontend: HTML + CSS + Jinja2

Графики: matplotlib → base64 PNG

Хранение: SQLite (локальная база для истории)

# Примеры использования
 Отслеживай свой прогресс после каждой игры
 
 Сравнивай любимых героев с друзьями
 
 Анализируй винрейт по дням
 
 Сохраняй историю поисков без регистрации

# Ограничения 
 OpenDota API имеет лимит ~60 запросов/минуту
 
 MMR отображается только если игрок играл в компетитивных матчах
 
 Steam ID должен быть в формате 64-bit (цифры)
